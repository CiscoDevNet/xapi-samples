    public connect(url: string, options: Object): XAPI source 

    import {connect} from 'jsxapi'

    Connect to an XAPI endpoint.

    Params:

    NameTypeAttributeDescription
    url string

    Connection specification.
    options Object
    • optional

    Connect options.
    options.host string
    • optional

    Hostname to connect to.
    options.username string
    • optional

    Username to authenticate with if protocol requires authentication.
    options.password string
    • optional

    Password to authenticate with if protocol requires authentication.
    options.loglevel string
    • optional

    Set the internal log level.

    Return:

    XAPI

    XAPI interface connected to the given URI.

    Example:

    const xapi = connect('ssh://host.example.com:22');

    public normalizePath(path: Array | string): Array source 

    import normalizePath from 'jsxapi/lib/xapi/normalizePath.js'

    Normalizes a path by turning it into an Array of strings. Removes empty parts of the path and ignores redundant whitespace or slashes. Each path element is also capitalized.

    Params:

    NameTypeAttributeDescription
    path Array | string

    Array or string path to normalize.

    Return:

    Array

    Array of path segments.
