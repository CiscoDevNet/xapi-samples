    XAPI

    Extends:

    events~EventEmitter → XAPI

    User-facing API towards the XAPI. Requires a backend for communicating with an XAPI instance. It should be possible to write backends for all kinds of transports (TSH over SSH, Websockets, HTTP, plain sockets, etc.)

    Example:

    Initialization
    const xapi = new XAPI(backend);
    Invoke commands
    xapi
  .command('Dial', { Number: 'johndoe@example.com' })
  .then(onSuccess, onFailure);
    Fetch a status
    xapi
  .status.get('Audio Volume')
  .then((value) => { console.log(value); });
    Set a configuration
    xapi
  .config.set('Audio DefaultVolume', 100);
    Listen to an event
    xapi.event.on('Some Event Name', (data) => {
  console.log(`Received feedback data: ${data}`);
});

    public command(path: Array | string, params: Object, body: string): Promise source

    Executes the command specified by the given path.

    Params:

    NameTypeAttributeDescription
    path Array | string

    Path to command node.
    params Object
    • optional

    Object containing named command arguments.
    body string
    • optional

    Multi-line body for commands requiring it.

    Return:

    Promise

    Resolved with the command response when ready.

    Example:

    // Space delimited
xapi.command('Presentation Start');

// Slash delimited
xapi.command('Presentation/Start');

// Array path
xapi.command(['Presentation', 'Start']);

// With parameters
xapi.command('Presentation Start', { PresentationSource: 1 });

// Multi-line
xapi.command('UserInterface Extensions Set', { ConfigId: 'example' }, `
 <Extensions>
   <Version>1.1</Version>
   <Panel item="1" maxOccurrence="n">
     <Icon>Lightbulb</Icon>
     <Type>Statusbar</Type>
     <Page item="1" maxOccurrence="n">
       <Name>Foo</Name>
       <Row item="1" maxOccurrence="n">
         <Name>Bar</Name>
         <Widget item="1" maxOccurrence="n">
           <WidgetId>widget_3</WidgetId>
           <Type>ToggleButton</Type>
         </Widget>
       </Row>
     </Page>
   </Panel>
 </Extensions>
`);
